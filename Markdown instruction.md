## Списки

Чтобы добавить ненумерование списки необходимо пункты выделить (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки необходимо просто пронумеровать. Напрмер, вот так:
1. Первый пункт
2. Второй пункт
